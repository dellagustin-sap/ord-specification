# ORD Interfaces

In this section you will find the technical ORD interfaces.

They are available as human readable documentation and also as machine-readable metadata.

Additionally, the ORD spec is also published as a NPM package, which includes both the JSON Schema interfaces and generated TypeScript definitions.

## ORD Version 1

- [ORD Document](../index.md#ord-document)
  - [Human readable documentation](Document.md)
  - [JSON Schema](https://open-resource-discovery.github.io/specification/spec-v1/interfaces/Document.schema.json)
  - [TypeScript Interfaces](https://open-resource-discovery.github.io/specification/spec-v1/interfaces/Document.ts.txt)
  - [CSV file](https://open-resource-discovery.github.io/specification/spec-v1/interfaces/Document.csv "Document.csv"), [Excel file](https://open-resource-discovery.github.io/specification/spec-v1/interfaces/Document.xlsx "Document.xlsx")
- [ORD Configuration](../index.md#ord-configuration-endpoint)
  - [Human readable documentation](Configuration.md)
  - [JSON Schema](https://open-resource-discovery.github.io/specification/spec-v1/interfaces/Configuration.schema.json)
  - [TypeScript Interfaces](https://open-resource-discovery.github.io/specification/spec-v1/interfaces/Configuration.ts.txt)
  - [CSV file](https://open-resource-discovery.github.io/specification/spec-v1/interfaces/Configuration.csv "Configuration.csv"), [Excel file](https://open-resource-discovery.github.io/specification/spec-v1/interfaces/Configuration.xlsx "Configuration.xlsx")
- [ORD Document Endpoint](../index.md#ord-document-endpoint) and [ORD Configuration Endpoint](../index.md#ord-configuration-endpoint)
  - [OpenAPI 3 Definition](./document-api.mdx)
